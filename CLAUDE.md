# 家庭小药箱

轻量、免费、可靠的本地健康档案管理工具

## 项目概述

- **类型**: 微信/支付宝小程序
- **技术栈**: Taro 4.x + React + TypeScript
- **目标用户**: 一家三口/四口的小家庭
- **核心功能**: 成员管理、生病记录、药品管理、数据迁移

## 设计文档

- `docs/plans/design.md` - 产品设计文档
- `docs/plans/implementation-plan.md` - 实现计划

## 关键约束

1. 无服务器 - 纯客户端应用，数据本地存储
2. 免费优先 - OCR使用免费额度
3. 简洁高效 - 页面简洁，交互直观
4. 用户主动 - 不推送提醒
